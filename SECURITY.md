# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 0.4.x   | ✅ Current release |
| < 0.4   | ❌ Not supported   |

## Reporting a Vulnerability

If you discover a security vulnerability in Prova, please report it responsibly:

1. **Do NOT** open a public GitHub issue.
2. Email the maintainer at the address listed on the [GitHub profile](https://github.com/Digvijay).
3. Include a detailed description of the vulnerability and steps to reproduce.
4. We will acknowledge your report within 48 hours and work toward a fix.

Thank you for helping keep Prova secure.
